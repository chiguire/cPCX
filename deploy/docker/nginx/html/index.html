DeerPost is a postcard exchange program for camp sites.

The fundamental idea is: for each postcard you send, you will receive one back from a random member in this event.

This website is heavily inspired by Postcrossing, which allows you to do the same with people from all over the world. Please do join Postcrossing if you like sending and receiving postcards!
